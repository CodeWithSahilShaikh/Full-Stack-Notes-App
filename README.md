# Full-Stack Notes App

A simple CRUD application for managing notes.

## Features
- Create new notes with title and description
- Edit existing notes
- Delete notes
- Responsive design

## Tech Stack
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Frontend:** HTML, CSS, JavaScript
